## Как добавить логотип

1. Поместите файл `logo.jpg` в корневую папку сайта (рядом с `index.html`).
2. Логотип автоматически появится в шапке и подвале.
3. Если нужно показывать изображение вместо текстового логотипа — откройте `css/style.css` и найдите класс `.logo-mark`, замените на тег `<img>` с `src="logo.jpg"`.

**Быстрая замена логотипа через поиск:**
Откройте любой HTML файл, найдите:
```html
<div class="logo-mark"><span>M</span></div>
```
Замените на:
```html
<img src="logo.jpg" class="logo-img" alt="Manas Stroi">
```

## Управление каталогом

### Через браузер (рекомендуется)
1. Откройте `admin.html`
2. Заполните форму: название, категория, цена, единица, описание, фото
3. Нажмите «Добавить материал»
4. Материал сразу появится в каталоге на сайте

Данные сохраняются в `localStorage` браузера и видны только на этом устройстве.

### Через JSON файл
Откройте `data/materials.json` и добавьте объект:
```json
{
  "id": 13,
  "name": "Название материала",
  "category": "facade",
  "price": "5000",
  "unit": "м²",
  "description": "Описание материала",
  "image": "путь/к/фото.jpg"
}
```

**Категории:** `facade` | `paving` | `travertine` | `polyfacade` | `other`

## Как запустить сайт

### Локально (для просмотра)
- Просто откройте `index.html` в браузере двойным кликом.
- Или используйте расширение Live Server в VS Code.

### На хостинге
1. Загрузите все файлы на хостинг через FTP или панель управления.
2. Убедитесь что `index.html` находится в корневой папке.
3. Сайт готов!

## Технологии

- **HTML5** — чистая разметка без фреймворков

- **JavaScript** — vanilla JS, без jQuery
